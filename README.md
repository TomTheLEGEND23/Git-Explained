# Git-Explained

Een praktisch leerpad voor het beheersen van **Git en GitHub**, gericht op commits, branches, remotes, merge en pull requests.

## Wat is Git-Explained?

Dit is een hands-on cursus waarin je stap voor stap leert:
1. **Commits** — kleine, duidelijke snapshots
2. **Branches** — veilig experimenteren
3. **Remotes** — samenwerken via GitHub
4. **Merge** — conflict-resolutie
5. **Pull Requests** — review en integratie

Alles gericht op **echte workflow** die je in elk team nodig hebt.

## 🚀 Aan de slag

### 1. Installeer Git en VS Code
- [Git for Windows](https://git-scm.com/install/windows)
- [Visual Studio Code](https://code.visualstudio.com/)

Bij Git-installatie: kies "Use Visual Studio Code as Git's default editor".

### 2. Fork en clone deze repository
- Klik rechtsboven **Fork** op GitHub.
- Je krijgt dan je eigen kopie naar `your-username/Git-Explained`.
- Clone je fork lokaal:
```bash
git clone https://github.com/your-username/Git-Explained.git
cd Git-Explained
```

## ⚠️ Belangrijke Afspraken
